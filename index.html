Как оформить заказ?

Внимание - минимальный заказ 2500 р. (исключение - доп заказ к уже отправленному). После нажатия на кнопку отправки заказа сначала появится окно для ввода дополнительной информации, в него будут входить: - Поле для ввода вашего имени (либо контактного лица, кто будет получать заказ). - Поле для ввода населенного пункта, куда должен товар доставляться. - Поле для ввода дополнительной информации к заказу. Эти поля НЕ ЯВЛЯЮТСЯ обязательными для заполнения. Чтобы отправить заказ менеджеру: 1. Нажмите «Копировать заказ» — текст скопируется в буфер обмена. 2. Вставьте заказ в чат с менеджеру (WhatsApp/Telegram) или отправьте на email: TOVAR-777@MAIL.RU.
